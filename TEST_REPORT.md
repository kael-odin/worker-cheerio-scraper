## 修复内容

### input_schema.json
- `linkSelector` 的 `editor` 从 `select` 改为 `input`（Cafe 不支持 select）

### main.js
- SDK 加载逻辑改为代理模式，自动检测环境：
  1. `global.cafesdk` 覆盖（测试脚本）
  2. `LOCAL_DEV=1` → 本地 SDK
  3. 默认 → 云 SDK（失败回退本地）
